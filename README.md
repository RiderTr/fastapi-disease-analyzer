# FastAPI Disease Analyzer

Цей проект є REST API додаток для аналізу великого набору медичних даних (U.S. Chronic Disease Indicators). Додаток дозволяє фільтрувати дані, знаходити статистику, завантажувати готові звіти у вигляді файлів та генерувати графіки захворюваності по штатах.

Як інтерфейс користувача (меню) використовується вбудована інтерактивна документація Swagger UI.

# Структура проекту
```
├── U.S._Chronic_Disease_Indicators.csv   <-- Датасет (потрібно завантажити і покласти сюди)
│
├── main.py          <-- Точка входа, настройка FastAPI и эндпоинты
├── data_loader.py   <-- Модуль завантаження, очищення даних та словники розшифровок
├── analytics.py     <-- Математична логіка, агрегація та пошук
├── report.py        <-- Генерація графіків (Matplotlib) та файлів для скачування
└── README.md        <-- Опис проекту
```

# Інструкція із запуску

## Крок 1: Підготовка оточення
Переконайтеся, що у вас встановлено Python (версія 3.8+). Встановіть необхідні бібліотеки за допомогою pip:
```
pip install fastapi uvicorn pandas matplotlib
```

## Крок 2: Підготовка даних
Помістіть у папку з кодом вихідний датасет. Файл має називатися строго U.S._Chronic_Disease_Indicators.csv.

## Крок 3: Запуск сервера
Відкрийте термінал у папці з проектом та виконайте команду:
```
uvicorn main:app --reload
```
Прапор --reload дозволяє серверу автоматично перезапускатись при зміні коду.

## Крок 4: Використання (Інтерфейс користувача)
Після успішного старту сервера відкрийте браузер і перейдіть за посиланням:
```
http://localhost:8000/docs
```
Ви побачите інтерактивне меню з усіма функціями:
- GET /topics - переглянути список всіх хвороб.
- GET /top-diseases - дізнатися топ найпоширеніших проблем зі здоров'ям.
- GET /search - знайти статистику за словом (наприклад, Asthma).
- GET /plot/states/{topic} - намалювати графік топ-10 штатів (введіть назву хвороби та натисніть Execute, картинка з'явиться нижче).
- GET /export/csv/{keyword} - завантажити звіт у форматі .csv на свій комп'ютер.
